\section{Coqによる$q$-類似の形式化}
\subsection{Coq}
Coq とは, 定理証明支援系の1つであり, 数学的な証明が正しいかどうか判定するプログラムである. 人間がチェックすることが難しい複雑な証明でも正しさが保証され, また証明付きプログラミングにも応用される. 例えば, 「$P \Lra Q$かつ$P$ならば$Q$」が常に正しいということは, Coq では
\begin{lstlisting}{Coq}
From mathcomp Require Import ssreflect.

Theorem modus_ponens (P Q : Prop) : (P -> Q) /\ P -> Q.
Proof.
  move=> [] pq p.
  by apply pq.
Qed.
\end{lstlisting}
と表現できる.
